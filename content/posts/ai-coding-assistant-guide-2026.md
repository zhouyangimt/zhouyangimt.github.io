## 2026年AI编程助手的格局
AI编程工具从2023年的「新鲜玩意」到2026年的「基础设施」，经历了爆发式增长。现在市面上的工具可以分为三类。
## 第一类：终端/CLI型
这类工具运行在终端里，直接操作代码库。
### Claude Code
- **厂商**：Anthropic- **特点**：200K上下文、理解力强、擅长架构级别的工作- **价格**：API计费，Max计划$200/月
### Codex (OpenAI)
- **厂商**：OpenAI- **特点**：轻量级、速度快、和GPT-4深度集成- **价格**：免费额度 + API计费
### Hermes Agent
- **厂商**：Nous Research（开源）- **特点**：完全开源、支持20+模型提供商、持久化记忆、技能系统- **价格**：免费（自带API Key）
## 第二类：IDE集成型

### Cursor
- **模式**：VS Code Fork- **杀手功能**：Tab预测补全、内联编辑- **价格**：免费版 + Pro $20/月
### Windsurf
- **模式**：独立IDE- **杀手功能**：Cascade多文件编辑、自动上下文- **价格**：免费版 + Pro $15/月
### GitHub Copilot
- **模式**：IDE插件- **杀手功能**：覆盖面最广、Agent模式- **价格**：$10/月（个人）
## 第三类：在线/Web型

### Bolt.new / v0.dev
- **功能**：一句话生成全栈应用- **适合**：快速原型、前端组件
## 本地 vs 云端：怎么选？
因素本地终端型IDE集成型Web型代码库理解⭐⭐⭐⭐⭐⭐编码速度⭐⭐⭐⭐⭐⭐架构/重构⭐⭐⭐⭐⭐⭐上手难度中低极低隐私好中差
## 我的工作流（供参考）
- **日常编码**：Cursor（Tab补全太香了）- **大型重构**：Claude Code（理解力天花板）- **自动化任务/定时任务**：Hermes Agent（开源 + cron）- **快速原型**：Bolt.new（最快从零到有）
## 最后建议
不要只用一个工具。组合使用，各取所长。我的成本：Cursor Pro $20 + Claude API ~$30 = 每月$50，换来的是生产力翻倍。
